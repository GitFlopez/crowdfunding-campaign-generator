# Crowdfunding Campaign Generator

> Generate complete Kickstarter, Indiegogo & GoFundMe campaign copy — page, rewards, backer emails, and social kit — in one session.

---

## The Problem

Most crowdfunding campaigns fail before they launch. Not because the product is bad — because the copy doesn't convert. The average funded campaign spends weeks agonizing over campaign page copy, reward tiers, and launch social posts. Most give up and publish something generic.

This skill generates everything: the campaign page story, reward tier copy, backer update emails, and a 23-piece social media launch kit — using the same frameworks used by campaigns that have raised $1M+.

## What You Get (4 Prompts)

| Prompt | Output | Length |
|--------|--------|--------|
| Campaign Page Copy | Hook, story, problem/solution, FAQ, CTA | 1,000–1,500 words |
| Reward Tier Descriptions | 6–8 tiers with names, pricing, urgency triggers | Full tier copy blocks |
| Backer Update Emails | 5-email sequence: pre-launch → thank you | Subject + body each |

## How to Use

1. Fill in the project brief (product name, platform, goal, story, team, traction)
2. Run Prompt 1 → paste into your campaign editor
3. Run Prompt 2 → build your reward tiers
4. Run Prompts 3 & 4 → schedule your emails and social posts
5. Launch

**Total time:** 45–60 minutes from blank page to ready-to-publish campaign
